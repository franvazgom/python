Múltiplos generales
================================================



Problema
--------

En el archivo `src\exercise.py` implementar una función con firma `multiplos(k:int, n:int)`:

* Debe retornar un conjunto con los múltiplos de `k` menores o iguales a `n`
* `k` $> 0$, `n` $\geq 0$
* `0` es múltiplo de `k`.
