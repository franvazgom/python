Múltiplos de 5
================================================

Problema
--------

En el archivo `src\exercise.py` implementar una función con firma `multiplos_de_5(a:int, b:int)`:

* Debe devolver un conjunto con los múltiplos de 5 en el rango $[a, b]$, incluyendo `a` y `b`.
* $1 \leq a \leq b$.
* `0` es múltiplo de 5.
